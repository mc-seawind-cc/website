## 2. 貼文風格指南

### 色調
- 延續網站風格：深色背景 + 藍紫色調
- 貼文圖片建議使用統一濾鏡或色調（海洋感、微冷色）
- 文字覆蓋使用 Noto Sans TC

### 標籤策略
**固定標籤（每篇都加）：**
```
#海風伺服器 #SeaWind #Minecraft伺服器 #Minecraft建築
```

**活動標籤（依活動加）：**
```
#海風珍奶日 #國際珍奶日 #Minecraft活動
```

### 語氣
- 親切、有溫度、像在跟朋友說話
- 適度使用 emoji 但不過度
- 可以用一些 Minecraft 玩家才懂的梗

## 3. 🧋 珍奶日貼文（貼文 #1）

### 📸 貼文圖片建議
**製作一張 1080×1080 或 1080×1350 的圖片，內容包含：**
- 深色背景（與網站風格一致，色號 `#0d1117`）
- 大標題：`🧋 海風國際珍奶日`
- 副標題：`4/27 – 5/31`
- 一排可愛的飲品 icon（13 種口味的小圖）
- 底部放海風 logo
- 可加入 CSS 風格的珍珠奶茶杯插畫（參考網站活動頁的 btea-cup 設計）

**配色參考：**
- 背景：`#0d1117` → `#161b22` 漸層
- 標題色：`#c49b6c`（金色/奶茶色）
- 副標色：`#9dafff`（天空藍）
- 強調色：`#a8e6cf`（薄荷綠）

### 📝 貼文文案

```
🧋 海風國際珍奶日來啦！

在海風，泡一杯屬於你的飲料 ☕✨

📅 活動期間：4/27 – 5/31
🍵 13 種限定口味飲品等你來收集
📦 14 種原料，任務＋自行採集雙管道
🪙 完成任務賺代幣，兌換限定物品
🗡️ 限定武器「珍奶微微」只送不賣！

四個階段：
🔹 調整期 4/27–5/3 → 任務開跑、囤原料
🔹 正式期 5/4–5/27 → 全口味開放、挑戰任務
🔹 兌換期 5/28–5/31 → 最後兌換、稱號結算

還有 4 個限定稱號等你解鎖 🏆
集齊 13 種飲品還能兌換「海風茶王」！

快來海風，一起乾杯吧 🥂

—

🌐 www.seawind.cc
💬 Discord 頻道連結在個人主頁
🎮 Java Edition 1.21+

#海風伺服器 #SeaWind #Minecraft伺服器
#海風珍奶日 #國際珍奶日 #珍奶日2026
#Minecraft活動 #Minecraft建築 #手搖飲
#珍珠奶茶 #BubbleTea #MinecraftJava
```

### 📌 貼文設定
- **格式**：單張圖片 或 輪播（Carousel，建議 3–5 張）
- **輪播建議頁面**：
  1. 主視覺（標題 + 日期 + 飲品圖）
  2. 玩法總覽（四步驟圖解）
  3. 飲品全覽（13 種口味小卡）
  4. 代幣商店 + 限定武器
  5. 稱號 + 活動時間軸
- **發佈時間建議**：活動前 2–3 天（4/24–4/25），晚上 8–10 點（台灣用戶活躍時段）
- **地標**：可加「台灣」或「Minecraft」相關地標
