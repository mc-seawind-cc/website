海風指南 › 領地系統

領地是海風的家園保護系統。建立領地後，別人不能在你的地盤搞破壞、偷東西或放置 TNT，你的建築和資源都會受到完整保護。

📋 什麼是領地？

你可以把領地想像成「虛擬圍籬」。在 Minecraft 裡，如果你不設領地，任何人都可以跑來你家拆牆、拿走箱子裡的東西。建立領地就像在地上畫一個保護圈，圈內的方塊只有你和你允許的人才能動。

領地系統，支援超過 110 種不同的權限設定（叫做 flags），從最基本的建造、破壞，到進階的 PVP、火焰蔓延、怪物生成都可以控制。

📌 基本資訊

項目限制
領地數量每分流各 10 個
子領地數量每分流各 50 個
長寬範圍500×500 格
高度範圍不限（地面到基岩層）
建立費用免費
有效期限永久（管理員刪除除外）
⚠️如果領地長期沒人使用（例如屋主很久沒上線），管理員有權將其刪除以釋放空間。建議保持活躍！

🔨 建立領地：手動框選法

最經典的方式，用木鋤自己選範圍，精確控制領地大小。

/res create <領地名稱>

  1. 拿一把 木鋤（在工作台上用木板 + 木棍合成）。
  2. 走到你想保護的區域一個角落，用木鋤 左鍵（預設挖掘鍵）點擊地面，設定第一個點。你會看到提示訊息。
  3. 走到對角的角落，用木鋤 右鍵（預設使用鍵）點擊地面，設定第二個點。
  4. 輸入 /res create 我的家（名稱可自訂），領地就建立完成了！
💡名稱只能用英文、數字和底線，不能用空格或中文。例如 my_home、base01 都可以。

⚡ 建立領地：自動框選法

不想跑來跑去？站在中心點一個指令搞定！

/res auto <領地名稱> [半徑]

  1. 站在你想建立領地的 正中心位置。
  2. 輸入指令，例如 /res auto 我的家。系統會自動以你為中心向外擴展。
ℹ️不指定半徑的話，預設會建立 500×384×500 的最大範圍領地。你也可以指定半徑，例如 /res auto 我的家 50 會建立 100×100 的範圍。

⚙️ 管理領地：設定權限

建立領地後，最重要的就是設定 flags（權限標誌）。Flags 決定了領地內允許或禁止哪些行為。

查看和設定通用權限

通用權限適用於所有人（非領地成員）：

/res set [領地名稱] <flag> [true/false/remove]

常用 flags 範例：

Flag作用建議值
build能否建造/破壞方塊false（禁止）
use能否使用門、按鈕、工作台等false（禁止）
container能否開箱子、桶子false（禁止）
pvp能否在玩家間戰鬥false（禁止）
firespread火焰能否蔓延false（禁止）
creeper苦力怕能否爆炸false（禁止）
tntTNT 能否爆炸false（禁止）
mobs怪物能否生成false（禁止）
move能否進入領地true（允許）

例如，要禁止所有人建造：/res set 我的家 build false

💡輸入 /res flags 可以查看完整的 flags 列表。

設定個別玩家權限

想讓朋友能進來但別人不行？用 /res pset 針對特定玩家設定：

/res pset [領地名稱] <玩家名稱> <flag> [true/false/remove]

例如：/res pset 我的家 Steve build true — 允許 Steve 在你的領地建造。

更簡單的方式是用 /res padd 直接給予玩家基本權限（建造、使用、開箱）：

/res padd [領地名稱] <玩家名稱>

移除玩家權限：/res pdel [領地名稱] <玩家名稱>

📐 擴縮領地

覺得領地太小或太大？可以用 expand 和 contract 調整大小。每次最多調整 100 格。

擴大領地

/res expand <格數>

  1. 面向你想擴大的方向（例如面向北邊就是往北擴）。
  2. 輸入指令，例如 /res expand 50 往前方擴大 50 格。

縮小領地

/res contract <格數>

  1. 背對你想縮小的方向（例如面向南邊就是從北邊縮）。
  2. 輸入指令，例如 /res contract 30 從背後方向縮小 30 格。

🏗️ 子領地

子領地（Subzone）就是在你的大領地裡面再畫一小塊，給那一小塊不同的權限。這功能叫做「zones within zones」。

常見用途：

  • 在自己的領地內設一個「公共區域」，讓所有人可以使用工作台。
  • 給朋友一個獨立的「房間」，只有他能在裡面建造。
  • 保護特定建築物，有不同的權限設定。

建立子領地的方式和建立一般領地一樣，只是在父領地的範圍內操作即可。每分流最多可以有 50 個子領地。

🗑️ 刪除領地

/res remove <領地名稱>

  1. 輸入指令，例如 /res remove 我的家。
  2. 系統會要求確認，再輸入 /res confirm 完成刪除。
⚠️刪除後無法復原！領地內的所有權限設定也會一併刪除。請謹慎操作。

📊 查看領地資訊

/res info [領地名稱] — 查看指定領地的詳細資訊（大小、擁有者、flags 等）。

/res list — 列出你擁有的所有領地。

/res tp <領地名稱> — 傳送到你的某個領地。

/res kick <玩家> — 把某人踢出你的領地。

/res ban <玩家> — 禁止某人進入你的領地。
