海風指南 › 公共傳送點

有些地點值得跟大家分享——像是礦場、刷怪塔、漂亮景點。公共傳送點讓你設好座標，全服玩家一鍵就能到。

請注意領地權限問題，避免在別人的領地上設傳送點造成糾紛。

💡此頁面不是個人傳送點。公共傳送點全服玩家都能使用。

📋 基本資訊

  • 每分流各 4 個額度，不跨分流共享。
  • 所有玩家都能看見和使用公共傳送點。
  • 設定者可以編輯和刪除自己建立的傳送點。

📖 使用教學

  • 設定公開傳送點 — 站在你想分享的位置，輸入 /warp set <名稱>。例如在漂亮景點輸入 /warp set 日落崖。
  • 傳送到公開點 — 輸入 /warp <名稱> 即可傳送。例如 /warp 日落崖。
  • 查看列表 — 輸入 /warp list 查看目前所有公共傳送點。
  • 刪除 — 自己設的傳送點可以用 /warp del <名稱> 移除。例如 /warp del 日落崖。

⚠️ 注意事項

  • 領地權限 — 請勿在他人領地內設定公共傳送點。如果傳送點設在沒有權限的地方，其他玩家可能無法正常使用。
  • 命名禮儀 — 名稱請取清楚好辨識的，不要用亂碼或容易混淆的名稱。
  • 分流限制 — 每個分流只有 4 個名額，用完就需要刪除舊的才能新增。

🔍 常用指令

💡提示：< > 為必選，[ ] 為可選。
指令說明

/warp <名稱>傳送至指定的公共傳送點
/warp list列出所有公共傳送點
/warp del <名稱>刪除指定的公共傳送點
