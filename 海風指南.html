快速開始

4 篇
🌊
新手指引
從零開始的海風生活：選單、簽到、建立據點、賺錢交易，一步步帶你融入伺服器
→
📱
伺服器選單
海風的控制中心，所有功能的入口。快捷鍵 /menu 或潛行+副手鍵開啟，裡面有快速傳送、購物、任務等分頁
→
🔗
綁定帳號
將 Minecraft 與 Discord 帳號綁定，解鎖跨平台通知、郵件查詢、驗證等功能。使用 /discord link 取得驗證碼後到 Discord 完成綁定
→
🗺️
線上地圖
即時世界地圖系統，邊走邊看位置自動更新。按 G 鍵或從選單「更多詳情」開啟，僅限 Java 版
→

領地與保護

3 篇
🏠
領地系統
你的家園圍籬。用木鋤選取範圍建立領地，別人就不能在你的地盤搞破壞或偷東西。支援權限管理、成員邀請、自訂規則設定
→
📋
互動記錄
查誰動了你的東西。點擊方塊即可查看最近 120 天的破壞、放置、開啟、取出等記錄，用 /co i 開關檢查器
→
🖼️
地圖署名
署名後的地圖別人無法複製，保護你的地圖畫作品。僅署名者可複製或解除署名
→

傳送系統

3 篇
📍
個人傳送點
回家、回礦場、回商店——設定 home 點走到哪傳到哪。全分流共 20 個額度，省下大量跑路時間
→
🌐
公開傳送點
把你的礦場、刷怪塔或漂亮景點分享給大家，其他玩家一鍵就能到。每分流各 4 個額度
→
🛗
電梯
蓋了高樓不用爬樓梯。垂直放置「浮雕石英方塊」作為樓層，按空白鍵上樓、蹲下鍵下樓
→

背包與倉庫

4 篇
🗄️
倉庫系統
背包塞滿了？全分流共 16 個大倉庫 + 8 個小倉庫，隨時存取物品，不用擔心放不下。用 /inv 開啟倉庫列表
→
🪟
玻璃展示櫃
把寶物放進透明展示櫃，可調整方向、大小或開啟自動旋轉，取代傳統盔甲座與展示框
→
🧍
盔甲座編輯器
自由編輯盔甲座的姿勢與裝備。到重生點「雲隱匠作」接取冒險任務即可獲得編輯器工具
→
🎒
快捷界伏盒
會員專屬的隨身額外背包，快速切換存取，背包管理更便利
→

任務與成長

3 篇
📜
任務系統
每日任務自動刷新、冒險任務找 NPC 接取、社區任務每天凌晨更新。完成任務獲得「任務幣」，可在兌換商店換取獎勵
→
📅
簽到系統
每天上線簽到領隨機獎勵。連續簽到有專屬稱號，累積簽到天數解鎖蛋糕類成就。特別的日子簽到或許有驚喜
→
🏅
稱號系統
顯示在名字旁邊的小標籤，展現你的身份與成就。有些稱號很稀有，是玩家之間互相認識的方式。用 /plt open 瀏覽、/plt shop 購買
→

經濟與交易

4 篇
💎
經濟系統
海風有三種貨幣：風幣（官方貨幣，鑽石 1:25 兌換）、任務幣（完成任務獲得）、冷媒（贊助回饋）。風汐系列物品可兌換特殊工具
→
🛒
線上商城
遊戲內直接買賣的交易平台，不用面對面交易。每分流 10 件上架額度，標好價格等生意上門
→
📦
箱子商店
最直覺的交易方式——放箱子、丟商品、設價格，其他玩家直接用箱子買。可設販賣或收購，每分流 120 個，適合蓋商店街
→
🏪
雜貨店
重生點 NPC 商店用風幣買基本材料。玩家也能在 Discord 論壇開自己的雜貨店，用 /sale 上架，買家點擊按鈕商品自動進背包
→

遊戲機制

12 篇
📖
機制總覽
海風獨有玩法的完整一覽：世界大小（±40 萬）、困難難度、建築高度 639 層、寶庫冷卻時間等核心設定
→
⛏️
連鎖挖掘
蹲下挖掘時自動連鎖破壞同類型方塊，挖礦砍樹效率倍增
→
💎
礦物代換
礦石隨機生成機制，分布與原版大致相同，但只有方塊曝露時才生成，維護經濟平衡
→
🔄
資源再生
7 天沒人造訪的區塊會自動重置，礦脈取之不盡。領地覆蓋的區塊不受影響，安心建設
→
⚒️
特殊合成
自訂合成配方：捕捉蛋分為「一次性」與「重複型」，直接合成無需權限，輕鬆獲得各類生物蛋
→
🪣
混凝土固化鍋
把混凝土粉末丟進裝水的鍋釜就能快速固化，不用再一塊塊倒水，也減少固化機造成的卡頓
→
📚
真實書櫃
點擊書櫃開啟介面，可以放入書本、書與筆等書類物品，打造真正的圖書館
→
🐝
蜂窩檢視
無模組玩家也能點擊蜂窩或蜂箱查看內部蜜蜂數量與蜂蜜飽滿程度
→
🚪
敲門與雙開
空手敲門發出敲門聲，點擊一扇門自動同步開啟兩扇。紅石也適用於雙開門機制
→
📦
穿牌開箱
點擊告示牌或展示框直接開啟相連的容器（箱子、木桶、界伏盒等），蹲下點擊進入編輯模式
→
🌍
地震預警
台灣地震頻繁，海風在遊戲聊天室設有即時地震通知。發生大地震請立即「趴下、掩護、穩住」
→
🪑
更多動作
點擊階梯或半磚直接坐下，點擊玩家可以坐在玩家身上，更多互動動作讓遊戲更有趣
→

社群互動

2 篇
📬
蜂聊郵政
不在線也能傳訊息或寄物品給朋友，每次寄件 1 元。補償、獎勵或遺失物領回也透過信箱系統處理
→
👻
靈魂出竅
進入自由觀察模式，穿牆飛行查看建築結構或觀察其他玩家，方便設計規劃或純粹閒逛
→

大廳區域

5 篇
🏘️
大廳簡介
重生點分為五大區域：重生點（格拉文特）、布雷特蘭、港口、小台北園區。每個區域都有 NPC、商店和冒險任務等你探索
→
🧱
建材小舖
販售各類建築木材與石料，包含著名特產「雲杉暖木」，也提供原木去皮服務。位於雪帆巷（45, 92, -67）
→
⚒️
雪隱匠作
重生點鐵匠鋪，可獲得盔甲座編輯器、雪山煤炭、邦布魔杖等特殊道具，也展示格拉文特土特產
→
🍵
風語茶屋
販售增益飲品與點心，提供「與自然對話」的飲茶生活方式。餅乾很好吃，真的。
→
🏦
銀行
格拉文特分行，使用鑽石兌換風幣（匯率 1:25）。位於原沫路上，潮沫巷左轉、風語茶屋旁
→

會員專屬

2 篇
🎒
快捷界伏盒
隨身額外背包空間，快速切換不同類型的物品儲存，告別背包塞滿的煩惱
→
🫥
隱藏裝備
隱藏身上裝備的外觀顯示，讓你的角色看起來更清爽，想秀什麼裝備由你決定
→
