海風指南›地圖

即時互動式世界地圖，在瀏覽器裡就能看到整個伺服器的地貌、領地和玩家位置。

📋 什麼是線上地圖？

海風使用線上地圖系統提供即時世界地圖。這就像是 Minecraft 世界的 Google 地圖，你可以在瀏覽器裡打開，看到整個伺服器的地形、建築、領地邊界，以及玩家目前的即時位置。

地圖會即時更新，當有人在蓋房子或挖礦時，地圖上也會同步顯示變化。這對於找路、查看領地範圍、或欣賞其他玩家的建築都很有幫助。

🚀 開啟方式

有兩種方式可以打開線上地圖：

方式操作
快捷鍵在遊戲內按下 G 鍵，系統會開啟瀏覽器並跳轉到地圖頁面。
伺服器選單輸入 /menu（或 /c、/dm）打開選單，點擊「更多詳情」進入地圖。
💡推薦使用 G 鍵快捷開啟，最方便！地圖會在你的預設瀏覽器中打開。

🗺️ 地圖功能

  • 即時地形：顯示整個世界的地貌、建築和地形變化，會即時同步更新。
  • 玩家標記：地圖上會標示所有在線玩家的即時位置和名稱，方便找人。
  • 領地顯示：領地的邊界會在地圖上以標記顯示，可以清楚看到哪些區域已被保護。
  • 縮放平移：可以用滑鼠滾輪縮放、拖曳平移，像操作 Google 地圖一樣直覺。

⚠️ 注意事項

⚠️線上地圖功能 僅限 Java 版 玩家使用。基岩版（手機、主機）玩家無法使用此功能。
  • 地圖是網頁版的，任何有瀏覽器的裝置都能查看（但遊戲內快捷鍵 G 僅限 Java 版）。
  • 地圖更新可能有幾秒到幾分鐘的延遲，不是完全即時的。
  • 未探索的區域不會顯示在地圖上（和遊戲機制一致）。
