⚙️ 權限

領地系統支援 114 種玩家可設定的權限，分為 8 大類。這裡是完整的權限參考表。

⚙️總計114 種
📂分類8 大類
🔧指令/res set

🔄 權限的三種狀態

每個權限都有三種狀態，你可以針對不同需求靈活設定：

✅

允許（true）

開放這項權限。例如設定 build true 就代表允許在領地內建造方塊。

❌

拒絕（false）

明確禁止這項行為。例如設定 container false 就代表禁止開啟箱子、熔爐等容器。

🔄

移除（remove）

恢復為預設值。不特別允許也不特別拒絕，交給系統預設處理。

🔧 設定方式

🌐

通用權限

/res set [領地] <權限> true/false
設定所有人適用的權限

👤

玩家權限

/res pset [領地] <玩家> <權限> true/false
設定特定玩家的權限

⚡

一鍵給予

/res padd [領地] <玩家>
一鍵給予常用權限（建造、使用、開箱等）

💡/res flags 查看完整列表。用 /res padd <玩家> 一鍵給予常用權限。
ℹ️PVP 保護：PVP 預設允許，建議設 /res set <領地> pvp false 關閉（含藥水效果）。

👥 設定個別玩家權限

想讓朋友能進來但別人不行？用 /res pset 針對特定玩家設定：

✏️

設定玩家權限

/res pset [領地] <玩家> <權限> [true/false/remove]

⚡

一鍵給予常用權限

/res padd [領地] <玩家>
自動給予建造、使用、開箱等基本權限

🗑️

移除玩家權限

/res pdel [領地] <玩家>
移除該玩家的所有特殊權限

/res set 領地名稱 權限名稱 是設定整個領地的權限（所有人都適用）；/res pset 領地名稱 玩家 權限名稱 是針對特定玩家設定。簡單說，set 是全領地規則，pset 是個人例外。
