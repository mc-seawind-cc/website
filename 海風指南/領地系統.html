海風指南 › 領地系統

領地是海風的家園保護系統。建立領地後，別人不能在你的地盤搞破壞、偷東西或放置 TNT，你的建築和資源都會受到完整保護。不管你是蓋了一棟夢想小屋、挖了一座礦場，還是和朋友一起建了一座城市，領地都能確保你的心血不被破壞。

海風的領地系統免費使用，支援所有世界（主世界、地獄、終界），建立後永久有效。搭配領地列表，你還可以瀏覽其他玩家的公開領地、留下評論、直接傳送過去參觀。

📋 什麼是領地？

你可以把領地想像成「保護區」。在海風，如果你不設領地，任何人都可以跑來你家拆牆、拿走箱子裡的東西。建立領地就像在區域裡面定一個保護區，區內的方塊只有你和你允許的人才能動。

領地系統支援超過 110 種不同的權限設定，從最基本的建造、破壞，到進階的 PVP、火焰蔓延、怪物生成都可以控制。你可以精細調整每個區域的權限——例如允許朋友在你的農場採收，但禁止他們開箱子；或者在公共區域開放使用工作台，但禁止放置方塊。

建立領地後，你還可以透過領地列表（/reslist open）瀏覽所有公開領地、查看評論、直接傳送。你可以把自己的領地設為公開，讓其他玩家來參觀、留言，甚至給你好評！

🔨 建立領地：手動框選法

最經典的方式，用木鋤自己選範圍，精確控制領地大小。選取時會有粒子效果顯示選取範圍，方便確認。

/res create <領地名稱>

  1. 拿一把 木鋤（在工作台上用木板 + 木棍合成）。
  2. 走到你想保護的區域一個角落，用木鋤 左鍵（預設挖掘鍵）點擊地面，設定第一個點。你會看到提示訊息。
  3. 走到對角的角落，用木鋤 右鍵（預設使用鍵）點擊地面，設定第二個點。
  4. 輸入 /res create 我的家（名稱可自訂），領地就建立完成了！
💡領地名稱支援英文、數字、中文、底線和連字號，最長 64 字元。例如 我的家、base01、海風小屋 都可以。
ℹ️選取時可以用 /res select vert 快速將選取範圍從腳下延伸到天空，省去手動調整高度的步驟。也可以用 /res select chunk 選取整個區塊（16×16）。

⚡ 建立領地：自動框選法

不想跑來跑去？站在中心點一個指令搞定！系統會自動偵測最佳範圍。

/res auto <領地名稱> [半徑]

  1. 站在你想建立領地的 正中心位置。
  2. 輸入指令，例如 /res auto 我的家。系統會自動以你為中心向外擴展。
  3. 系統會偵測是否與其他領地重疊，如果有衝突會提示你調整位置。
ℹ️不指定半徑的話，預設會建立最大範圍的領地。你也可以指定半徑，例如 /res auto 我的家 50 會建立以你為中心、半徑 50 格的範圍。如果同名領地已存在，系統會自動在名稱後加上編號（例如「我的家的領地1」）。

⚙️ 管理領地：設定權限

建立領地後，最重要的就是設定 flags（權限標誌）。Flags 決定了領地內允許或禁止哪些行為。

查看和設定通用權限

通用權限適用於所有人（非領地成員）：

/res set [領地名稱] <flag> [true/false/remove]

常用 flags 範例：

Flag作用建議值
build能否建造/破壞方塊。false（禁止）
use能否使用門、按鈕、工作台等。false（禁止）
container能否開箱子、桶子。false（禁止）
pvp能否在玩家間戰鬥。false（禁止）
firespread火焰能否蔓延。false（禁止）
creeper苦力怕能否爆炸。false（禁止）
tntTNT 能否爆炸。false（禁止）
mobs怪物能否生成。false（禁止）
move能否進入領地。true（允許）

例如，要禁止所有人建造：/res set 我的家 build false

💡輸入 /res flags 可以查看完整的 flags 列表。海風支援超過 110 種不同的 flags，從基本的建造、使用，到進階的火焰蔓延、怪物生成、PVP 等都可以控制。
ℹ️PVP 保護：在領地內，如果 PVP flag 設為 false（預設），玩家之間無法互相傷害。包含藥水效果（如虛弱、中毒）也會被阻擋。

設定個別玩家權限

想讓朋友能進來但別人不行？用 /res pset 針對特定玩家設定：

/res pset [領地名稱] <玩家名稱> <flag> [true/false/remove]

例如：/res pset 我的家 Steve build true — 允許 Steve 在你的領地建造。

更簡單的方式是用 /res padd 直接給予玩家基本權限（建造、使用、開箱）：

/res padd [領地名稱] <玩家名稱>

移除玩家權限：/res pdel [領地名稱] <玩家名稱>

💬 訊息設定

領地有專屬的進出訊息功能。當玩家進入或離開你的領地時，聊天欄會自動顯示你設定的訊息。

  • 進入訊息：/res message <領地名稱> enter <訊息>
  • 離開訊息：/res message <領地名稱> leave <訊息>
💡訊息支援顏色代碼，例如 &a 是綠色、&c 是紅色。可以用 %player 顯示進入的玩家名稱、%residence 顯示領地名稱。

📐 擴縮領地

覺得領地太小或太大？可以用 expand 和 contract 調整大小。每次最多調整 100 格。

擴大領地

/res expand <格數> [方向]

  1. 面向你想擴大的方向（例如面向北邊就是往北擴），或在指令中指定方向（up/down/north/south/east/west）。
  2. 輸入指令，例如 /res expand 50 往前方擴大 50 格，或 /res expand 10 up 往上擴大 10 格。

縮小領地

/res contract <格數> [方向]

  1. 背對你想縮小的方向（例如面向南邊就是從北邊縮），或在指令中指定方向。
  2. 輸入指令，例如 /res contract 30 從背後方向縮小 30 格。

🏗️ 子領地

子領地（Subzone）就是在你的大領地裡面再畫一小塊，給那一小塊不同的權限。這功能叫做「zones within zones」。

常見用途：

  • 在自己的領地內設一個「公共區域」，讓所有人可以使用工作台。
  • 給朋友一個獨立的「房間」，只有他能在裡面建造。
  • 保護特定建築物，有不同的權限設定。

建立子領地的方式和建立一般領地一樣，只是在父領地的範圍內操作即可。每分流最多可以有 50 個子領地。

🗑️ 刪除領地

/res remove <領地名稱>

  1. 輸入指令，例如 /res remove 我的家。
  2. 系統會要求確認，再輸入 /res confirm 完成刪除。
⚠️刪除後無法復原！領地內的所有權限設定也會一併刪除。請謹慎操作。

📊 查看領地資訊

/res info [領地名稱] — 查看指定領地的詳細資訊（大小、擁有者、flags 等）。

/res list — 列出你擁有的所有領地。

/res tp <領地名稱> — 傳送到你的某個領地（有 3 秒傳送延遲）。

/res kick <玩家> — 把某人踢出你的領地。

/res ban <玩家> — 禁止某人進入你的領地。

/res unstuck — 如果被困在某個領地出不來，用這個指令脫困。

📋 領地列表與評論

海風有專屬的領地列表系統，讓你用介面瀏覽、搜尋和管理所有領地。輸入 /reslist open 即可開啟。

瀏覽公開領地

所有設為「公開」的領地都會出現在列表中。你可以：

  • 查看資訊 — 點擊領地查看擁有者、大小、成員數、評論等詳細資訊
  • 直接傳送 — 右鍵點擊公開領地即可傳送過去（需有傳送權限）
  • 釘選收藏 — 丟棄鍵（Q）釘選常用領地，方便快速找到
  • 排序篩選 — 依照名稱、評論數或大小排序，支援升降序切換

評論系統

你可以對公開領地留下評論，幫助其他玩家了解這個領地：

1

開啟領地資訊

在領地列表中點擊想評論的領地

2

選擇好評或差評

左鍵給予好評、右鍵給予差評，並在聊天欄輸入評論內容

3

完成評論

評論會顯示在領地資訊中，所有玩家都能看到

管理自己的領地

領地擁有者可以透過列表管理自己的領地：

  • 修改名稱 — 左鍵點擊「標示」按鈕，在聊天欄輸入新名稱（最長 16 字元）
  • 修改描述 — 右鍵點擊「標示」按鈕，輸入領地描述（支援換行 \n）
  • 更換圖標 — 蹲下+左鍵點擊「標示」按鈕，將背包物品拖曳至指定位置
  • 公開/私人 — 切換領地是否出現在公開列表中
  • 設定傳送點 — 右鍵「傳送」按鈕，將目前站立位置設為領地傳送點
💡領地圖標會顯示在列表中，選擇一個有辨識度的物品當圖標，讓別人更容易認出你的領地。部分物品（如箱子、鎖鏈、紅石）不能作為圖標。
