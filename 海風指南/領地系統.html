🛡️ 領地系統

用木鋤圈一塊地，守護你的心血。免費、永久、支援所有世界。

🏠上限每分流 10 個
📏最大範圍500×500 格
💰費用免費

在海風，不設領地就等於把家門敞開。建立領地後，別人不能在你的地盤搞破壞、偷東西或放置 TNT，你的建築和資源都會受到完整保護。不管你是蓋了一棟夢想小屋、挖了一座礦場，還是和朋友一起建了一座城市，領地都能確保你的心血不被破壞。

🏠 什麼是領地？

你可以把領地想像成「保護區」。在海風，如果你不設領地，任何人都可以跑來你家拆牆、拿走箱子裡的東西。建立領地後，區內的方塊只有你和你允許的人才能動。

🏠

領地的運作方式

用木鋤框出兩個對角點（外圈範圍），建立成功後整塊區域變成你的領地。只有你和你允許的人能在裡面建造、破壞或開箱子。你還可以在領地內再畫一小塊「子領地」，給不同人不同的權限。

你的領地（受保護） 公共區域

領地系統支援多種玩家可設定的權限，從最基本的建造、破壞，到進階的 PVP、火焰蔓延、怪物生成都可以控制。你可以精細調整每個區域的權限——例如允許朋友在你的農場採收，但禁止他們開箱子。

建立領地後，你還可以透過領地列表（/reslist open）瀏覽所有公開領地、查看評論、直接傳送。你可以把自己的領地設為公開，讓其他玩家來參觀、留言、給你好評！

🏗️ 建立領地

海風提供兩種建立領地的方式，選一個適合你的：

精確控制

🔨 手動框選法

用木鋤選兩個對角點，精確控制領地大小。選取時會有粒子效果顯示範圍。

1 在工作台用木材＋木棍合成木鋤。
2 左鍵點方塊設定第一個角。
3 走到對角，右鍵點方塊設定第二個角。
4 輸入 /res create <名稱>。
快速方便

⚡ 自動框選法

不想跑來跑去？站在中心點一個指令搞定，系統自動偵測範圍和衝突。

1 站在你想保護的區域正中心。
2 輸入 /res auto <名稱>。
3 系統偵測衝突後自動建立完成。
💡可加半徑：/res auto 我的家 50。
💡名稱支援英文、數字、中文、底線和連字號，最長 64 字元。快捷指令：/res select vert 垂直延伸到天空、/res select chunk 選取整個區塊（16×16）。

📐 建立子領地

子領地就是在你的大領地裡面再畫一小塊，讓那小塊擁有獨立的權限設定。建立方式和上面完全一樣，只是必須在主領地範圍內操作。子領地會繼承父領地的預設權限，但可以自行覆蓋任何設定。

💡快捷方式：站在子領地中心點，輸入 /res create 子領地名稱，系統會自動以你為中心框出預設大小的範圍。

⚙️ 領地權限

建立領地後，下一步就是設定權限——決定誰能在你的地盤上做什麼。預設情況下，外人不能建造、不能開箱子，也不能開門或使用工作台。如果要讓朋友使用這些功能，就需要開放權限。

三種設定方式

根據情境選擇最適合的方式：

🌐

通用權限（所有人）

設定「預設權限」——套用到所有非成員玩家。輸入指令後會跳出一個勾選清單介面，所有權限分類列出，直接點選要開放或關閉的項目即可。

/res set — 設定你站的這塊領地

/res set 領地名稱 — 設定指定領地

👤

玩家權限（指定玩家）

幫特定玩家設定專屬權限，會覆蓋通用設定。例如通用設定關閉建造，但你可以單獨允許某個朋友建造。介面操作方式和通用權限一樣。

/res pset 玩家名 — 設定該玩家在你站的領地的權限

/res pset 領地名稱 玩家名 — 設定指定領地

⚡

修改基本權限（快速給予）

一個指令同時開放多項常用權限，適合讓朋友快速入住。開放的權限包含：建造（build）、使用（use）、開箱（container）、開門（door）、紅石（redstone）等基本互動。

/res padd 玩家名 — 給予基本權限

/res pdel 玩家名 — 移除基本權限

💡指令都要站在領地內才有效。如果不站在領地內，需要在指令後加上領地名稱（例如 /res set 我的家）。

實際操作流程

完整流程

⚡ 讓朋友能使用你的領地

1 站在你的領地內（重要！指令需要知道你在哪塊領地）。
2 輸入 /res pset 冒險者，會打開權限設定介面，直接勾選要開放的權限即可。
3 如果只想快速開放基本權限（建造、開箱、開門等），用 /res padd 冒險者 一個指令搞定。
4 如果之後想移除，輸入 /res pdel 冒險者 即可。
💡進階技巧：/res padd 和 /res pset 可以混用。先用 padd 快速開放基本權限，再用 pset 微調——例如先 padd 給朋友全部權限，再用 /res pset 冒險者 把 PVP 關掉。

📐 擴縮領地

覺得領地太小或太大？可以用 expand 和 contract 調整大小。每次最多調整 100 格。

📏 擴大領地

/res expand <格數> [方向]

1 面向你想擴大的方向。
2 輸入指令，例如 /res expand 50 往前方擴大 50 格。

📐 縮小領地

/res contract <格數> [方向]

1 背對你想縮小的方向。
2 輸入指令，例如 /res contract 30 從背後方向縮小 30 格。

🗑️ 刪除領地

/res remove <領地名稱>

  1. 輸入指令，例如 /res remove 我的家
  2. 系統會要求確認，再輸入 /res confirm 完成刪除
⚠️刪除後無法復原！領地內的所有權限設定也會一併刪除。請謹慎操作。

🏗️ 子領地

子領地就是在你的大領地裡面再畫一小塊，讓那小塊擁有獨立的權限設定。你可以把子領地想成「領地中的領地」——大範圍保護整個基地，小範圍精細控制每個房間。子領地會繼承父領地的預設權限，但可以自行覆蓋任何設定。

💡 常見用途

子領地的應用非常靈活，以下是最常見的三種情境：

🤝

公共區域

在領地內設一塊「公共區域」，讓所有人可以使用工作台、熔爐或交易，但不能破壞其他東西。

🚪

朋友房間

給朋友一個獨立的房間，只有他能在裡面建造和放置方塊，但不能動領地其他地方。

🏰

分區保護

花園、倉庫、紅石區各自獨立設定，不同區域不同權限，管理更精細。

📐 嵌套規則

子領地可以繼續往下細分，但有幾個限制要注意：

📏

範圍限制

子領地不能超出父領地的範圍，必須完全包含在內。

🔄

權限繼承

子領地會繼承父領地的預設權限，但可以自行覆蓋任何設定。

📦

數量限制

每個主領地最多 50 個子領地，每個子領地還能再往下細分（最多 3 層）。

🏷️

命名規則

子領地名稱在同一個父領地內不能重複，支援中文命名。

💬 訊息設定

領地有專屬的進出訊息功能。當玩家進入或離開你的領地時，聊天欄會自動顯示你設定的訊息。

  • 進入訊息：/res message <領地名稱> enter <訊息>
  • 離開訊息：/res message <領地名稱> leave <訊息>
💡訊息支援顏色代碼，例如 &a 是綠色、&c 是紅色。可以用 %player% 顯示進入的玩家名稱、%residence% 顯示領地名稱。
