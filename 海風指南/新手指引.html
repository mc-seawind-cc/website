剛加入海風不知道從哪開始？這篇指南會帶你一步步上手，從連線、生存、建家到賺錢交易，讓你輕鬆融入海風的生活。

🌊 歡迎來到海風伺服器

海風是一個休閒生存伺服器。在這裡，你可以慢慢建造自己的據點、研究紅石機器、種田釣魚、探索世界，或逛逛其他玩家的家園。

我們希望提供一個「輕鬆、友善、自由」的遊戲環境，讓玩家能長久地在這個世界遊玩。

伺服器在保留原版體驗的基礎上，也加入了一些輔助與趣味系統，讓遊玩更加便利。

🔌 如何加入海風

項目資訊
伺服器位址seawind.cc
推薦版本1.21.11
相容版本1.21.6 ～ 1.21.x（皆可進入）
  1. 開啟 Minecraft 啟動器，選擇 1.21.11 版本（或 1.21.6 以上的版本）。
  2. 進入「多人遊戲」→「新增伺服器」，位址填入 seawind.cc。
  3. 儲存後雙擊伺服器即可加入！
💡如果連不上，請確認版本是否正確，或檢查防火牆是否阻擋了 Minecraft 的連線。

🏘️ 出生點小鎮

第一次進入海風，你會出現在一座小鎮。這是伺服器的中心區域，有各種 NPC、商店和設施可以逛逛。

不過，小鎮只是起點——想要真正開始生存，你需要離開小鎮，到外面的世界建立自己的據點。

ℹ️小鎮是安全區域，不會被破壞。可以先四處逛逛熟悉環境，再出發探險。

🚀 開始你的海風生活

ℹ️Minecraft 新手可至 Minecraft Wiki 查看新手手冊。
  1. 離開小鎮 — 在聊天欄輸入 /rtp 隨機傳送到世界各地，尋找你喜歡的環境建立據點。平原、山區、海邊、叢林……選一個你心動的地方！
  2. 建立據點 — 蓋一座屬於你的家。初期先做個小屋、放個箱子和床就夠了，之後可以慢慢擴建。
  3. 保護你的家 — 蓋好後立刻使用領地系統保護你的土地，避免被陌生人破壞或竊取物品。這是最重要的一步！
  4. 設定傳送點 — 透過個人傳送點建立傳送點，方便快速往返小鎮和據點。
  5. 學會使用選單 — 輸入 /menu 或潛行鍵 + 副手鍵開啟選單，大部分的功能都可以在選單中找到。
  6. 每日簽到與任務 — 每日可透過簽到與完成任務獲得風幣與風汐碎片，並有機會得到特殊獎勵。

⌨️ 必備指令速查

以下是你在海風最常用到的指令，建議先記住前幾個：

指令用途說明
/menu開啟選單大部分功能的入口，也可以潛行+副手鍵開啟
/rtp隨機傳送隨機傳送到世界某處，適合找建家地點
/spawn回到出生點回到小鎮中心
/tpa <玩家>傳送請求向其他玩家發送傳送請求
/tpahere <玩家>邀請傳送邀請其他玩家傳送到你身邊
/sethome <名稱>設定家園在目前位置設定個人傳送點
/home <名稱>回家傳送回你設定的家園
/res create <名稱>建立領地保護你的土地，詳見領地系統
/bal查看餘額查看你擁有的風幣數量
/pay <玩家> <金額>轉帳把風幣轉給其他玩家
/discord link綁定 Discord將 MC 帳號與 Discord 綁定，詳見綁定帳號
💡忘記指令沒關係，輸入 /menu 開啟選單，大多數功能都可以用點的！

🛡️ 領地保護——最重要的一件事

在海風，沒有保護的土地任何人都可以進出。如果你不想辛苦蓋的家被陌生人翻箱倒櫃，建好據點後第一件事就是圈地。

  1. 準備一把木鏟（可在選單或小鎮取得）。
  2. 走到你想保護的區域的一個角落，左鍵點擊地面，記錄第一個座標。
  3. 走到對角的另一個角落，右鍵點擊地面，記錄第二個座標。
  4. 輸入 /res create <領地名稱>，完成圈地！

詳細的領地指令與進階設定，請見領地系統完整指南。

⚠️圈地需要花費風幣。初期先圈一塊小範圍就夠了，之後有錢再擴大。詳見經濟系統了解如何賺取風幣。

💰 經濟入門——風幣是什麼

風幣是海風的通用貨幣，用來圈地、交易、購買物品等。以下是新手賺取風幣的方式：

方式說明
每日簽到每天簽到即可獲得風幣，連續簽到有額外獎勵
完成任務系統會發送每日任務，完成可獲得風幣與風汐碎片
銀行兌換在小鎮的銀行使用鑽石兌換風幣
箱子商店在市場區開設自己的商店，賣東西給其他玩家

更多經濟細節請見經濟系統。

❓ 常見問題

❓連不上伺服器怎麼辦？
確認版本是 1.21.6 以上（推薦 1.21.11），位址是 seawind.cc。如果還是不行，檢查防火牆或嘗試重新啟動遊戲。
❓在小鎮卡住了怎麼辦？
輸入 /spawn 回到小鎮中心。如果在野外迷路，也可以用 /spawn 回到出生點再重新出發。
❓不小心死了，東西怎麼辦？
海風有背包備份系統，可以向管理員申請回滾。在 Discord 問題頻道開單說明情況即可。
❓怎麼找到其他玩家？
使用 /tpa <玩家名稱> 發送傳送請求，對方同意後就會傳送到他們身邊。也可以在線上地圖查看其他玩家的大致位置。
❓風幣不夠用怎麼辦？
每天簽到 + 做任務是最穩定的收入來源。也可以在市場區開店賣東西，或到銀行用鑽石兌換。詳見經濟系統。

🗺️ 探索海風世界

海風的世界廣闊而充滿驚喜，每一個角落都值得慢慢探索。無論是喜歡冒險、建設還是收集資源，這裡都有屬於你的天地。

從小鎮開始探索，到處走走逛逛，遇到居民時左鍵右鍵都試著點點看，或許可以發現冒險任務！完成任務有時候會獲得特殊道具。在野外也會遇到各種有趣的地點和建築，慢慢享受探索的樂趣吧。
