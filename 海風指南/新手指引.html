🌊 新手指引

第一次加入海風？別擔心，這篇指南會在 5 分鐘內帶你從零開始，到擁有自己的家園。

📡位址seawind.cc
🎮版本建議 1.21.11
⚔️難度困難
🌍世界±40 萬格

🌊 關於海風

海風是一個以建築養老、慢節奏生存為核心的 Minecraft 社區伺服器。這裡沒有 PvP 強迫、沒有怪物攻城，你可以安心蓋房子、經營自己的世界。

我們的理念很簡單：

  • 免費功能多 — 領地（最大 500×500，最多 10 塊）、倉庫（16 大＋8 小）、任務、稱號、箱子商店、傳送、連鎖挖礦，全部免費開放。這些功能在其他伺服器通常需要贊助才能使用
  • 不刪檔、不跑路 — 長期穩定營運，你的努力不會白費
  • 透明公正 — 管理團隊玩家意見即時回饋，共建海風文化
  • 中文化介面 — 精心設計的中文選單，新手也能輕鬆上手，無需額外安裝模組
ℹ️想了解更多海風的歷史與文化，歡迎逛逛文化藝廊和歷史館。

🔌 30 秒完成連線

只要三步，就能進入海風的世界。

seawind.cc
🟢 Java 版，建議使用 1.21.11 連線（1.21.6 以上皆可）
📱 基岩版最新版，埠號 51496
1

選擇版本

開啟 Minecraft 啟動器，建議選擇 1.21.11 連線（1.21.6 以上版本皆可進入）。推薦使用 Java 版電腦遊玩，體驗最完整。

2

新增伺服器

進入「多人遊戲」→「新增伺服器」，位址填入 seawind.cc。基岩版在「遊戲」→「新增伺服器」，位址填 seawind.cc，埠號填 51496。

3

加入！

儲存後雙擊伺服器，等待連線完成就可以開始冒險了

💡推薦使用 Java 版電腦遊玩，手機/主機的基岩版也可以進入，但部分功能可能有差異。連不上請確認版本是否正確，或檢查防火牆。

⛪ 出生點：格拉文特

第一次進入海風，你會出現在格拉文特正中心的扶搖禮拜堂。格拉文特是海風的重生點區域，坐落在一座山谷中（海拔約 y=90）。

大廳區域不只有格拉文特——附近還有嵐峰鎮、布雷特蘭、港口和小台北園區。其中格拉文特和嵐峰鎮有各自的 NPC、商店和冒險任務。

不過，這裡只是起點——大廳區域非常廣闊（約 ±4000 ～ ±7000 格），走是走不出去的。想要真正開始生存，你需要用指令離開大廳，到外面的世界建立自己的據點。

ℹ️格拉文特是安全區域，不會被破壞。這裡保留原始生態，沒辦法挖木頭或開採資源，也沒有開啟領地飛行。扶搖禮拜堂周圍有物捐庫、雪隱匠作、風語茶屋等設施，建材小舖在北邊。遇到 NPC 可以左鍵右鍵都試著點點看。

想了解更多格拉文特的故事，請見文化藝廊 — 格拉文特。

🚀 你的第一天

做好準備了嗎？跟著以下步驟，一步步建立你的海風生活。

1

離開重生點，找到你喜歡的地方

在聊天欄輸入 /rtp 隨機傳送到世界各地。大廳區域走不出去（範圍約 ±4000 ～ ±7000 格），必須用指令離開。平原、山區、海邊、叢林……選一個你心動的地方！不需要一次就找到完美的地點，先安定下來再說。

2

建立你的第一個據點

蓋一座屬於你的家。初期先做個小屋、放個箱子和床就夠了，之後可以慢慢擴建。木屋、石屋、洞穴基地——隨你發揮。

3

🏠 立刻保護你家！

這是最重要的一步！不是用盾牌，而是用領地系統圈地。根據社群須知，無領地區域通常視為公共資源，雖然管理員會依情況判斷處理，但有系統會在每 7 天後重置無領地區塊。蓋好後馬上圈起來。詳見下方「領地保護」。

4

設定傳送方式

在海風，回到據點有兩種方式：/sethome 家 設定個人傳送點，之後 /home 家 隨時回來；或者在領地裡輸入 /res tp 領地名 也能直接傳送。建議兩種都設好，方便又保險。詳見個人傳送點。

5

開啟選單，探索所有功能

輸入 /menu 或按住潛行鍵 + 副手鍵開啟選單。海風的大部分功能都可以在選單中找到——購物、任務、傳送、設定，一個入口搞定。

6

每日簽到與任務

每天上線記得簽到和做任務，這是穩定獲得風幣和獎勵的方式。簽到還有連續天數稱號可以拿！

ℹ️Minecraft 新手可至 Minecraft Wiki 查看新手手冊。

🛡️ 領地保護——最重要的一件事

在海風，無領地區域通常視為公共資源（見社群須知第一章第1條）。雖然管理員會依情況判斷處理，但有系統會在每 7 天後重置無領地區塊。建好據點後第一件事就是圈地。

1

合成木鋤

用木板和木棍合成一把木鋤，作為圈地工具

2

選取角落

走到區域一角，左鍵點擊地面

3

選取對角

走到對角另一角，右鍵點擊地面

4

建立領地

輸入 /res create 名稱，完成！

💡圈地不用花錢，每個領地最大 500×500 格，最多可建立 10 塊領地——比多數伺服器慷慨得多（一般伺服器通常只給 2～3 塊、100～250 格大小）。初期先圈一塊小範圍就夠了，之後再慢慢擴大或新增。

完整的領地指令與進階設定（權限管理、成員邀請、自訂規則），請見領地系統完整指南。

💬 社交與互動

海風不只是單機生存，和其他玩家互動會讓遊戲更有趣。

💬 聊天

遊戲內直接打字就能和附近的人聊天。想私訊某人可以用 /msg <玩家> 訊息。

📦 交易

可以面對面交易，也可以用箱子商店或線上商城掛賣。風幣是通用貨幣。

🏠 社區

海風有社區系統，可以和志同道合的玩家一起建設、共享資源。詳見社區系統。

💡遇到 NPC 記得左鍵右鍵都點點看，很多隱藏對話和功能！

🎉 活動與節慶

海風定期舉辦各種活動，從節慶限定到建築比賽都有，參加還能獲得限定稱號和獎勵！

🧋 限時活動

配合節日舉辦的限定活動，例如珍奶日、端午節、萬聖節等，有專屬 NPC、道具和稱號。

📸 社區活動

攝影大賽、建築比賽等社區活動，展現你的創意，還有獎勵可以拿。

⚔️ 常駐玩法

風汐系列、那個魚等常駐活動隨時都能玩。

最新活動資訊請見活動頁面。

🆘 遇到問題怎麼辦？

玩遊戲難免遇到問題，以下是求助管道：

📋 Discord 客服單

到海風 Discord 開客服單，管理員會協助處理問題。包含舉報違規、申請回報、物品問題等。

📢 問管理員

在遊戲內看到管理員可以直接詢問。管理員 ID 前通常有標記，或查看海風團隊名單。

📖 查指南

你正在看的這份海風指南涵蓋了大部分系統與功能，善用搜尋功能快速找到答案。
